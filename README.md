# 🌐 Cloud Resume Challenge – Frontend

**Cloud Resume Challenge frontend static site** — onderdeel van mijn cloud-leertraject.

🔗 Live demo: https://www.dvandijk.com

---

## 📌 Over dit project

Dit is het frontend deel van mijn Cloud Resume Challenge project — een statische website die fungeert als persoonlijke portfolio en interactieve bezoekerteller.

De site is gebouwd met HTML, CSS en JavaScript en wordt gehost via Azure Blob Storage Static Website.

Het doel van dit project is:
- Praktische ervaring opdoen met frontend technologieën (HTML, CSS, JS)
- Integratie met een serverless backend API
- Werken met Azure cloud services voor hosting en storage

---

## 🧠 Functionaliteit

- 📄 Statische HTML site met persoonlijke informatie  
- 📊 Dynamische *visitor counter* via een backend API  
- 💡 Responsive design voor mobiel en desktop  
- ☁️ Gehost als static website in Azure Blob Storage  

> Het bestand `script.js` stuurt requests naar een backend API die de visitor counter bijwerkt en retourneert.

---

## 🛠️ Gebruikte technologieën

- 🧱 HTML
- 🎨 CSS
- 📟 JavaScript
- ☁️ Azure Blob Storage — Static Website Hosting
- ⚡ Azure Functions — Backend API
- 🗄️ Azure Cosmos DB — Visitor data storage
- 🌐 Cloudflare — DNS, CDN & caching layer
